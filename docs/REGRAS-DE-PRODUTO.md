# Regras de Produto

- Não prometer dinheiro fácil.
- Não comparar usuários.
- Não criar ranking público.
- Não usar linguagem de vergonha.
- Se o usuário perder sequência, mostrar: "Tudo bem. Recomece hoje com uma ação simples."
- IA personalizada fica guardada para o plano Premium futuro e não será lançada agora.
- Basic acessa o app normal sem IA personalizada.
- Admin acessa painel de usuários e pode ajustar plano/status.
- Dados financeiros pertencem ao usuário autenticado.
- Cada usuário só pode ver os próprios dados.
- localStorage não é banco principal.
- Navegação principal deve ter poucas abas e rolagem vertical.
- O lançamento deve aceitar fala ou escrita.
- O Índice da Virada mede evolução, não riqueza.
