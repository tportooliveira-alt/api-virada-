# Roadmap

## Entregue nesta evolução

- Base Supabase Auth e PostgreSQL
- Migration SQL com tabelas públicas e RLS
- Perfis com planos `basic`, `premium` e papel `admin`
- Quatro abas principais mobile-first
- Lançamento por fala ou escrita
- Parser local `parseFinancialInput`
- Índice da Virada
- Pontos e medalhas
- Modo anti-impulso
- PWA com manifesto
- Admin inicial de usuários
- Estrutura de IA premium guardada para ativação futura, sem lançamento agora

## Próximos passos

- Criar projeto Supabase real e aplicar migrations
- Testar cadastro/login com confirmação de email configurada
- Validar RLS com dois usuários reais
- Melhorar parser com mais categorias e sinônimos brasileiros
- Ativar IA premium somente quando o produto decidir lançar esse plano
- Integrar pagamentos para upgrade premium
- Criar service worker customizado se for necessário cache offline avançado
