# Histórico de Decisões

## 2026-04-28

- Decisão final de usar Supabase como backend e banco principal.
- localStorage deixa de ser banco principal e fica reservado para cache temporário.
- IA personalizada fica guardada para o plano Premium futuro, sem lançamento agora.
- Navegação principal reduzida para quatro áreas: Início, Lançar, Evolução e Aprender.
- Lançamentos podem ser feitos falando ou escrevendo.
- App deve ser PWA instalável no celular.
- Sistema de Pontos de Virada e medalhas incluído.
- Índice da Virada mede comportamento e evolução, não riqueza.
- Dados são separados por usuário com Row Level Security.
- Admin pode ver usuários e ajustar plano/status manualmente.
