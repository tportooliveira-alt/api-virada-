# Contexto do Projeto

O Virada App é o app profissional ligado ao produto Código da Virada Financeira.

Promessa central: Não é dinheiro fácil. É clareza, organização, plano, ideias práticas e direção.

O app atende brasileiros endividados ou desorganizados financeiramente que precisam registrar entradas, gastos, dívidas, metas, missões e decisões de compra com o mínimo de fricção.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase Auth
- Supabase PostgreSQL
- Row Level Security
- PWA instalável
- Deploy futuro na Vercel

## Áreas Principais
